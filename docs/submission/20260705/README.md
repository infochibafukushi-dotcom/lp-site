### 印刷方法

1. 管理画面の「提出セット一括PDFを開く」、または `pre-fixed-fare-submission-full-set-v1-candidate.pdf` をブラウザで開く
2. ブラウザの印刷画面を開く（Ctrl+P / ⌘+P）
3. 用紙サイズを **A4** に設定する
4. 部数を **3部** に指定する（関東運輸局手引きの正・副・控に相当）
5. 印刷またはPDF保存を実行する

※ 3部ぶん同じ内容をPDF内で3回連結する方式は採用していません。印刷ダイアログの部数指定で3部印刷してください。

※ 個別PDF 6件は引き続き本フォルダに保管してください。一式PDFは印刷用の結合版です。final化は正式提出日前に別途行います。

### 読みやすさ改善（2026/07/05）

運輸局への紙提出・説明を想定し、**ページ数削減より読みやすさを優先**する印刷用レイアウト基準を適用しています。

- フォントサイズ基準：h1 18pt / h2 16pt / h3 13pt / 本文 11pt / 表 9.5〜10pt / 注記 9pt（8.5pt未満は不使用）
- ページ余白：`@page margin 16mm 14mm 20mm 14mm`（下余白 20mm でフッター重なりを防止）
- 小分類・見出し+本文・表行・画面キャプチャ+説明は `break-inside: avoid` で途中分断を抑制
- 別紙セットの「画面キャプチャ貼付資料」に検証用4画面の実画像を掲載（空欄貼付欄を廃止）
- 統合説明資料の画面キャプチャ章も実画像4枚を掲載
- 一式PDFの総ページ数は読みやすさ優先のため固定しません（最新再生成時: **80ページ**）
